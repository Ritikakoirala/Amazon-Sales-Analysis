Amazon Sales Dataset Analysis
Project Overview

This project involves the analysis of an Amazon Sales Dataset to extract insights, trends, and patterns in product sales. The goal is to perform Exploratory Data Analysis (EDA), visualize trends, and identify factors affecting sales performance. This dataset can be useful for e-commerce analysts, business strategists, and data enthusiasts who want to understand online retail behavior.

Objectives

Data Cleaning: Handle missing values, inconsistent data, and duplicates.

Exploratory Data Analysis (EDA): Understand data distribution, identify top-selling products and categories.

Sales Trend Analysis: Analyze seasonal trends, monthly/yearly sales patterns.

Visualization: Represent sales trends using charts and plots.

Insights Extraction: Identify factors that impact sales performance and customer buying behavior.
